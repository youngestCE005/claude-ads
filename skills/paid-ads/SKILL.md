# Paid Ads — Meta Campaign Strategy & Structure

Produces a complete campaign brief and Ads Manager launch specification for
Meta (Facebook / Instagram). Covers audience targeting, budget, bid strategy,
ad set structure, naming conventions, and a pre-launch checklist.

## Process

1. Collect inputs: offer description, target audience, objective, budget
2. Identify business type (e-commerce, lead gen, SaaS, local service, etc.)
3. Generate **Campaign Brief** (Step 1)
4. Generate **Campaign Structure Doc** (Step 2)
5. Write brief to `campaigns/{name}/brief.md`
6. Write structure to `campaigns/{name}/structure.md`
7. Initialize `campaigns/{name}/log.md` with creation entry

## Step 1 — Campaign Brief

Produce a structured brief covering:

### Offer Analysis
- Product/service description and positioning angle
- Price point and value proposition
- Offer mechanism (lead magnet, demo, free trial, discount)
- Funnel entry point (where does this ad lead?)

### Audience Breakdown
- Primary audience: job titles, company size, geography, interests, behaviors
- Estimated Meta audience size for each segment
- Recommended ad set split: prospecting vs. retargeting

### Budget Recommendation
- Daily spend per ad set (minimum: 5x target CPA for learning phase exit)
- Minimum run duration (recommend 14 days before optimization)
- CBO vs. ABO recommendation with rationale
- Budget allocation: prospecting % / retargeting %

### KPI Targets
Set platform-appropriate benchmarks for:
- CPL (Cost per Lead) or CPA (Cost per Acquisition)
- CTR (click-through rate): target ≥1.0%
- Lead form completion rate: target ≥15%
- Frequency (7-day): keep <3.0 for prospecting

## Step 2 — Campaign Structure Doc

Produce the full Ads Manager specification:

### Campaign Settings
```
Campaign name: {client}_{objective}_{audience}_{YYYY-MM}
Objective: {Leads | Sales | Traffic | Awareness}
Budget type: CBO (preferred) or ABO with rationale
Special Ad Category: {None | Housing | Credit | Employment | Social Issues}
```

### Ad Set Specifications
For each ad set (minimum 3 recommended):

```
Ad Set name: {audience-segment}_{placement}
Audience: {targeting parameters}
Placement: Advantage+ Placements (default) or manual spec
Budget: ${daily} / day
Bid strategy: {Lowest Cost | Cost Cap | Bid Cap}
Schedule: {start date} — {end date or ongoing}
```

Targeting parameters include:
- Age range, gender (if relevant)
- Locations (countries, regions, or cities)
- Interests and behaviors (with Meta audience size estimate)
- Custom Audiences or Lookalike Audiences
- Exclusions (existing customers for prospecting, etc.)

### Ad Specifications
Three ads per ad set across three creative angles:

| Ad Name | Angle | Format | Headline | CTA |
|---------|-------|--------|----------|-----|
| `pain_image_v1` | Pain | Single image | {≤40 chars} | {CTA} |
| `outcome_image_v1` | Outcome | Single image | {≤40 chars} | {CTA} |
| `proof_image_v1` | Social Proof | Single image | {≤40 chars} | {CTA} |

### Lead Form Specification (if Lead Gen objective)
- Form name and type (More Volume vs. Higher Intent)
- Questions: name, email, phone (standard) + 1-2 qualifying questions max
- Thank-you message and link
- Privacy policy URL

### Campaign Folder Structure
```
campaigns/{campaign-name}/
  brief.md
  structure.md
  copy.md          ← generated by /ad-creative
  images/          ← generated by /html-ad
  tracking.md      ← generated by /analytics-tracking
  ab-test.md       ← generated by /ab-test-setup
  report.md        ← post-launch
  log.md
```

## Pre-Launch Checklist

Before publishing, verify:

- [ ] Meta Pixel installed and firing on destination page
- [ ] Conversions API (CAPI) active — reduces iOS 14.5 signal loss
- [ ] Conversion event selected matches campaign objective
- [ ] UTM parameters added to all destination URLs
- [ ] Ad creative meets Meta's 20% text rule (use Text Overlay tool)
- [ ] All images are 1080×1080px or 1080×1350px
- [ ] Lead form tested in preview
- [ ] Budget is at least 5x target CPA per ad set per day
- [ ] Audience overlap check run (Audience Overlap tool in Ads Manager)
- [ ] Naming conventions followed across campaign, ad sets, and ads
- [ ] Special Ad Category declared if applicable
- [ ] Campaign set to "Off" until final review complete

## Output

Write the following files to `campaigns/{campaign-name}/`:
- `brief.md` — complete campaign brief
- `structure.md` — full Ads Manager spec
- `log.md` — initialized with creation timestamp

Then confirm: "Campaign brief and structure saved to `campaigns/{campaign-name}/`. Next: run `/ad-creative` to generate copy, then `/html-ad` for static ad images."
